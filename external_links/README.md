# RobotoEmpireBot (Telegram)

## Quick Start

1. Open Telegram and talk to @BotFather.
2. Create a new bot:
   - Use /newbot
   - Name: RobotoEmpireBot
   - Username: @roboempirebot
3. Copy the token and paste it into:
   - R:\Repos\Roboto-SAI-2026\secrets\telegram_token.txt
4. Run the deploy script:
   - R:\Repos\Roboto-SAI-2026\external_links\telegram_deploy.bat
5. Search for @roboempirebot in Telegram and send:
   - /start

## Commands

- /start: Activate the empire greeting
- /status: Show MCP status list
- /conquer: Receive a conquest haiku

## Auto-start (Hidden)

Double-click:

- R:\Repos\Roboto-SAI-2026\external_links\auto_start.vbs

This runs the bot in the background.
